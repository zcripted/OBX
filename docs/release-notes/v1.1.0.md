# OBX v1.1.0

## Economy & Shop

* New quantity-based shop menu with buy/sell modes and purchase confirmation.
* Dynamic pricing and stock limits with automatic restocking.
* Fully functional in-game Shop Editor (`/shop admin`).
* New Sell Wand for instant chest selling.
* New Bank system with GUI, transaction history, and interest tiers.
* Server Account for taxes, fees, and other economy sinks.
* Daily sell limits, rank sell multipliers, and economy audit logs.
* New Economy Control Panel for administrators.
* Auction House improvements, fixes, and new commands.

## Backpack

* New portable Backpack system (`/backpack`).
* Convert backpacks into physical items.
* Dupe-proof token protection with backpack recovery.
* Backpack nesting prevention.

## Admin & Server Management

* New `/health` command for server diagnostics.
* Improved `/obx` diagnostics and troubleshooting.
* Redesigned console logging and startup banner.
* Better plugin list formatting in console.

## Chat & Messaging

* Fixed chat prompt messages leaking into public chat.
* Improved private message reply buttons.
* Better name and message formatting protection.

## Update Notifications

* Staff update notifications enabled by default.
* Automatic startup and periodic update checks.
* Clickable download and release notes links.
* Improved notification formatting.

## Fixes & Improvements

* Fixed `/unbreakable` on all supported versions.
* Improved Folia compatibility and thread safety.
* Multiple backpack duplication and data-loss fixes.
* Fixed teleport warmup movement cancellation.
* Fixed combat state, warp prompt, jail duration, and reload edge cases.
* Improved caching and database performance.
* Better support for legacy 1.8 servers.

## Security

* Removed sensitive webhook data from default configs.
* Added webhook configuration warnings.
* Fixed permission-related startup crash.

## Internationalization

* Full English, German, and Spanish support for all new features.

## Compatibility

* Supports Minecraft 1.8.8+ through modern Paper/Folia versions.
